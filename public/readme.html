全方位 AI 數位內容創作夥伴

這是一個為數位內容創作者深度訂製的高效再製工具。不論您的原始素材是影片字幕 (SRT 檔) 或是逐字稿文稿，本工具皆能在數秒內一鍵幫您重整並擴展為符合 SEO 的部落格文章、多平台社群貼文、行銷電子報、社群輪播圖結構與資訊圖表規劃，為您省去 90% 以上的手工轉換時間！

本程式使用 Google 頂級 Gemini 大語言模型，藉由精心調校的 System Prompts，確保產出品質專業、結構完整，且帶有高度可讀性。

七大核心分頁功能

🎙️

語音轉文字辨識 (Tab 0)

支援上傳多種格式的影音檔案，透過 Gemini AI 或是免費部署您的專屬 Cloudflare Whisper Worker，快速且高精準地將語音內容轉換為純文本與 SRT 字幕。

📝

智慧字幕整理 (Tab 1)

可點擊或拖曳匯入 SRT 字幕檔。具備自動句末標點補全、錯別字智慧修正、長句自動分拆等功能。提供四個精準控制滑桿，並可一鍵取得 AI 影片摘要與精緻章節報告。

✍️

部落格文章生成 (Tab 2)

將整理後的逐字稿重構為結構完整、排版優良的部落格草稿。內建「寫作客製化風格精靈」設定語氣、字數與視角。支援進階 SEO 關鍵字分析、SEO 標題推薦與內部連結建議，並搭載 Quill 富文本編輯器。

📱

多平台社群貼文 (Tab 3)

針對 Facebook、Instagram、Threads、LinkedIn、LINE 等平台量身打造吸引點擊的文案。提供多種寫作風格（如懸疑提問、直白痛點、金句破題），自動生成熱門 Hashtags 與行動呼籲 (CTA)。

✉️

行銷電子報 (Tab 4)

將長文精煉為具備高度點擊開信率的 HTML 電子報。系統將自動為您生成 3 組具誘因的郵件主旨、簡短預覽文字，以及精心排版的行銷郵件 HTML 原始碼。

🎨

社群輪播圖規劃 (Tab 5)

專為 IG / FB 多圖滑動輪播打造。自動規劃每一頁輪播圖的視覺畫面構圖與文字標題，並針對 Stable Diffusion 或 Midjourney 等 AI 繪圖工具提供最合適的中英文繪圖提示詞 (Prompts)。

📊

資訊圖表規劃 (Tab 6)

分析文稿或文章，規劃出適合製作成單張大圖資訊圖表 (Infographic) 的結構報告。自動提供圖表推薦指示、排版結構分析，以及視覺設計的中英文提示詞 (Prompts)。

快速上手指南

1

設定 Gemini API 金鑰

點選首頁狀態列旁的「設定金鑰」或側邊欄底部的「金鑰管理」按鈕，展開「📖 如何免費申請？」連結。依照教學，登入 Google 帳號並至 Google AI Studio 點擊 "Create API Key" 免費申請，將取得以 AIza 開頭的金鑰複製，貼回本工具儲存即可。本工具亦支援填入多組金鑰實現負載平衡輪替。

2

匯入並一鍵整理字幕檔 (Tab 1)

您可以至 DownSub 等網站下載您的 YouTube 影片 SRT 字幕檔。在 Tab 1 將字幕內容直接貼入或拖曳上傳至虛線匯入區。調整右側控制面板的「每行字數」、「最小合併字數」、「時間平移」、「間隔修復」滑桿，最後點選「智慧重整字幕」。完成後，可在此分頁直接點擊「生成摘要」與「生成章節」取得 AI 初步精簡報告。

3

一鍵生成部落格文章 (Tab 2)

整理好文稿後，前往 Tab 2 部落格分頁。首選左側設定區的「最佳化文稿以提升寫作品質」，能大幅提升後續生成文章的流暢度與正確性。點選「風格客製化精靈」微調人設（如第一人稱、第三方導師）、字數與語氣。設定完成點擊「開始撰寫文章」，即可在右側編輯器獲得專業草稿，並支援 SEO 關鍵字與內部連結分析。

4

多軌擴展社群與視覺規劃 (Tab 3 - 6)

您可以根據剛產出的部落格文章或原始整理稿，切換至 Tab 3「社群貼文」、Tab 4「行銷電子報」、Tab 5「輪播圖提示詞」或 Tab 6「資訊圖表規劃」，一鍵生成不同平台的宣傳內容，形成全網覆蓋的數位內容行銷矩陣。

如何取得 Cloudflare Worker 網址與 Token (進階語音辨識)

為了使用更精準的 Whisper 語音辨識引擎 (Tab 0)，您需要部署專屬的 Cloudflare Worker 來作為 API 的中繼站。這不僅免費，且能確保您的辨識資料安全不外洩。請依循以下步驟進行設定：

1

部署專屬的 Worker

在本專案的 cf-worker 目錄下，包含了一個預設好的腳本。請確保您已安裝 npm。開啟終端機進入該目錄後，執行 npm install，然後執行 npx wrangler deploy。系統會自動引導您登入 Cloudflare 帳號並完成部署。

2

取得 Worker 網址 (URL)

部署成功後，Wrangler 會在終端機輸出您的專屬網址，格式類似 https://your-worker.your-subdomain.workers.dev。請將這段網址**完整複製**，並進入 Lumina 程式左下角的「設定金鑰」選單中，貼上至「Whisper Worker 網址」欄位。

3

設定驗證金鑰 (Token)

為了保護您的 Worker 不被他人濫用，您必須設定一組自訂的 Token。請進入您的 Cloudflare 儀表板，找到剛剛部署的 Worker -> Settings -> Variables and Secrets。新增一個名為 API_TOKEN 的環境變數，值設定為您自訂的一串密碼 (例如：my-secret-token-123)。

最後，將這串自訂的密碼貼回 Lumina 程式內的「Whisper 驗證金鑰 (Token)」欄位並儲存，即可開始享受快速且免費的高品質語音辨識服務！

常見問題與進階設定

Q1: 免費申請的 Gemini API Key 有限制嗎？

目前 Google AI Studio 提供的免費金鑰 (Gemini 2.5 Flash) 有每分鐘 15 次請求的限制。若您需要處理極大量的影片或文章，建議依照本程式金鑰設定提示，申請 2 到 3 組不同的免費金鑰一同填入，系統會自動在呼叫時進行智慧輪替，避免超出請求上限限制。

Q2: 程式有關閉頁面即清除金鑰的機制嗎？

有的。在您設定金鑰時，可以下拉選擇「金鑰保存期限」。如果您是在公用電腦操作，建議選擇「分頁關閉即清除」(存於 Session Storage)，當您將此瀏覽器分頁關閉後，所有 API 金鑰即會被自動銷毀抹除，無須擔心外洩。

Q3: 草稿與編輯中的資料會遺失嗎？

本程式內建即時草稿自動保存技術 (Auto-Save)。您的字幕內容、部落格優化文章與 SEO 設定將自動儲存於瀏覽器快取中。當您不小心重新整理頁面或斷電重開時，歡迎首頁會動態出現「繼續上次編輯」按鈕，點擊即可一鍵回復上次的工作區狀態。

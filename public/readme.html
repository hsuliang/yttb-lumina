全方位 AI 數位內容創作夥伴

這是一個為數位內容創作者深度訂製的高效再製工具。不論您的原始素材是影片字幕 (SRT 檔) 或是逐字稿文稿，本工具皆能在數秒內一鍵幫您重整並擴展為符合 SEO 的部落格文章、多平台社群貼文、行銷電子報、社群輪播圖結構與資訊圖表規劃，為您省去 90% 以上的手工轉換時間！

本程式使用 Google 頂級 Gemini 大語言模型，藉由精心調校的 System Prompts，確保產出品質專業、結構完整，且帶有高度可讀性。

七大核心分頁功能

🎙️

語音轉文字辨識

支援上傳多種格式的影音檔案，透過 Gemini AI 或是免費部署您的專屬 Cloudflare Whisper Worker，快速且高精準地將語音內容轉換為純文本與 SRT 字幕。

📝

智慧字幕整理

可點擊或拖曳匯入 SRT 字幕檔。具備自動句末標點補全、錯別字智慧修正、長句自動分拆等功能。提供四個精準控制滑桿，並可一鍵取得 AI 影片摘要與精緻章節報告。

✍️

部落格文章生成

將整理後的逐字稿重構為結構完整、排版優良的部落格草稿。內建「寫作客製化風格精靈」設定語氣、字數與視角。支援進階 SEO 關鍵字分析、SEO 標題推薦與內部連結建議，並搭載 Quill 富文本編輯器。

📱

多平台社群貼文

針對 Facebook、Instagram、Threads、LinkedIn、LINE 等平台量身打造吸引點擊的文案。提供多種寫作風格（如懸疑提問、直白痛點、金句破題），自動生成熱門 Hashtags 與行動呼籲 (CTA)。

✉️

行銷電子報

將長文精煉為具備高度點擊開信率的 HTML 電子報。系統將自動為您生成 3 組具誘因的郵件主旨、簡短預覽文字，以及精心排版的行銷郵件 HTML 原始碼。

社群輪播圖規劃

專為 IG / FB 多圖滑動輪播打造。自動規劃每一頁輪播圖的視覺畫面構圖與文字標題，並針對 Stable Diffusion 或 Midjourney 等 AI 繪圖工具提供最合適的中英文繪圖提示詞 (Prompts)。

📊

資訊圖表規劃

分析文稿或文章，規劃出適合製作成單張大圖資訊圖表 (Infographic) 的結構報告。自動提供圖表推薦指示、排版結構分析，以及視覺設計的中英文提示詞 (Prompts)。

快速上手指南

1

設定 Gemini API 金鑰

點選首頁狀態列旁的「設定金鑰」或側邊欄底部的「金鑰管理」按鈕，展開「📖 如何免費申請？」連結。依照教學，登入 Google 帳號並至 Google AI Studio 點擊 "Create API Key" 免費申請，將取得以 AIza 開頭的金鑰複製，貼回本工具儲存即可。本工具亦支援填入多組金鑰實現負載平衡輪替。

2

匯入並一鍵整理字幕檔

您可以至 DownSub 等網站下載您的 YouTube 影片 SRT 字幕檔。在「智慧字幕整理」分頁將字幕內容直接貼入或拖曳上傳至虛線匯入區。調整右側控制面板的「每行字數」、「最小合併字數」、「時間平移」、「間隔修復」滑桿，最後點選「智慧重整字幕」。完成後，可在此分頁直接點擊「生成摘要」與「生成章節」取得 AI 初步精簡報告。

3

一鍵生成部落格文章

整理好文稿後，前往「部落格文章生成」分頁。首選左側設定區的「最佳化文稿以提升寫作品質」，能大幅提升後續生成文章的流暢度與正確性。點選「風格客製化精靈」微調人設（如第一人稱、第三方導師）、字數與語氣。設定完成點擊「開始撰寫文章」，即可在右側編輯器獲得專業草稿，並支援 SEO 關鍵字與內部連結分析。

4

多軌擴展社群與視覺規劃

您可以根據剛產出的部落格文章或原始整理稿，切換至「社群貼文」、「行銷電子報」、「社群輪播圖規劃」或「資訊圖表規劃」分頁，一鍵生成不同平台的宣傳內容，形成全網覆蓋的數位內容行銷矩陣。

關於 Cloudflare Whisper 運作原理與額度限制

⚙️ 它是如何運作的？

與傳統「將大檔案上傳到單一伺服器慢慢排隊處理」不同，Cloudflare 擁有遍佈全球的邊緣運算網路（Serverless GPU）。當您在本程式上傳長影音時，程式會在您的電腦端**自動將音檔切成一小塊一小塊的碎片**（例如每段幾十秒），並同時發送到您專屬的 Cloudflare Worker。
這時，Cloudflare 全球的伺服器會「平行處理」這些碎片，再將辨識好的文字瞬間拼裝回來還給您。這也是為什麼它的速度可以極快，且您無需擔心隱私外洩（因為資料只在您自己的帳號內流轉）。

💰 免費額度與限制

部署 Cloudflare Worker 本身是免費的，而使用其內建的 AI 模型（Workers AI）也提供了非常慷慨的每日免費配額：

  • 每日免費計算量：Cloudflare 每天提供 10,000 個「神經元 (Neurons)」的免費運算量。
  • 換算實際長度：對於 Whisper 語音辨識模型而言，這大約足夠您**每天免費轉譯數小時**長度的音檔。
  • 超量怎麼辦？即使您某天處理了超量影片，只要您沒在 Cloudflare 綁定信用卡開啟付費方案，系統只會拒絕請求而**絕對不會擅自向您收費**；若您未來選擇綁定信用卡，超出的部分收費也極度低廉（處理數十小時往往只需不到一美金）。

💡 結論：對於一般創作者日常的 YouTube 影片、Podcast 逐字稿製作，Cloudflare 的免費額度幾乎等於「無限制」，加上完全掌握在自己手裡的隱私權，是一項極高性價比的隱藏神兵利器。

如何部署專屬的 Cloudflare Whisper Worker

為了使用更精準的 Whisper 語音辨識引擎，您需要部署專屬的 Cloudflare Worker 來作為 API 的中繼站。這不僅免費，且能確保您的辨識資料安全不外洩。請依循以下步驟從頭部署：

1

建立新的 Cloudflare Worker

請前往 Cloudflare 儀表板 並登入。在左側選單點選 Workers & Pages，接著點選 Create application -> Create Worker -> Start with Hello World!。

2

設定 Worker 網址

點選 Worker name ，填入你要的網址，這是 Worker API URL，用來填寫連線設定。填好後按下 Deploy。

3

貼上並部署程式碼

部署完成後，點選該 Worker 的 Edit code（或 Quick Edit）。將本專案目錄下 cf-worker/worker.js 的所有程式碼複製，並覆蓋掉編輯器內的預設程式碼。完成後，點擊右上角的 Save and Deploy。

4

啟用 Workers AI Binding (⚠️ 非常重要)

回到該 Worker 的設定頁面，點選畫面上方 Bindings，然後點擊 Add binding。選擇 Workers AI，並在 Variable name 欄位輸入 AI (必須全大寫且完全一致)。點擊 Save 儲存。

5

取得網址並設定驗證金鑰 (API Token)

為了保護 Worker 不被盜用，建議到 Settings -> Variables 頁面，新增一個環境變數：Type 選擇 Secret，Variable name 填寫 API_TOKEN，Value 填寫您自訂的密碼（例如 my-secret-token-2026）並點選 Deploy 儲存。

最後，請回到 ㄚ亮笑長的內容助手 程式左下角的「設定金鑰」選單，將您的 Worker 網址 (例如 https://yttb-whisper.your-name.workers.dev) 與剛剛設定的 Token 填入「Worker 連線設定」頁籤中並儲存，即可開始享受快速免費的高品質語音辨識服務！

常見問題與進階設定

Q1: 免費申請的 Gemini API Key 有限制嗎？

目前 Google AI Studio 提供的免費金鑰 (Gemini 2.5 Flash) 有每分鐘 15 次請求的限制。若您需要處理極大量的影片或文章，建議依照本程式金鑰設定提示，申請 2 到 3 組不同的免費金鑰一同填入，系統會自動在呼叫時進行智慧輪替，避免超出請求上限限制。

Q2: 程式有關閉頁面即清除金鑰的機制嗎？

有的。在您設定金鑰時，可以下拉選擇「金鑰保存期限」。如果您是在公用電腦操作，建議選擇「分頁關閉即清除」(存於 Session Storage)，當您將此瀏覽器分頁關閉後，所有 API 金鑰即會被自動銷毀抹除，無須擔心外洩。

Q3: 草稿與編輯中的資料會遺失嗎？

本程式內建即時草稿自動保存技術 (Auto-Save)。您的字幕內容、部落格優化文章與 SEO 設定將自動儲存於瀏覽器快取中。當您不小心重新整理頁面或斷電重開時，歡迎首頁會動態出現「繼續上次編輯」按鈕，點擊即可一鍵回復上次的工作區狀態。
